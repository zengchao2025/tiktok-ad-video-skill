# TikTok Ad Video Skill

**一套可以直接喂给 AI 的 TikTok 广告视频生成技能包。**

> 别再自己憋 Prompt 了。把这里的文档原样丢给 DeepSeek、ChatGPT 或任何大模型，它就能按专业标准为你写出可直接用于视频生成平台的高质量 Prompt。

---

## 这个项目怎么用？

整个项目的核心使用方式只有一步：

1.  **选择文档**  
    根据你要用的视频生成平台选择对应的技能文档：
    -   通用平台 / 标准流程 → 复制 `HappyHorse CreativeOS v2.1-MVP.md` 的全部内容
    -   需 Seedance 格式资产输出 → 复制 `Seedance.md` 的全部内容

2.  **丢给 AI**  
    把文档内容粘贴到 DeepSeek（或 ChatGPT、Claude 等）的对话框中。可以配合这句话：

    > “请以上述文档作为你的系统指令和唯一规范，现在我要开始生成 TikTok 广告视频 Prompt。”

3.  **直接提需求**  
    AI 会自动按照四元公式、视觉增强层、评分体系等内部规则，为你生成结构完整、可审计的 Prompt。你只需要描述产品、场景或创意方向即可。

**底层逻辑：** 两份 `.md` 文件就是两个完整的 System Prompt，分别封装了 Prompt 工程规范、质量门控、运镜语法和审计闭环。AI 读完之后就会“变成”一个专业的广告 Prompt 工程师。

---

## 📦 你得到的是什么？

-   **专业级 Prompt**：包含精确词数控制、运镜语言、视觉增强词（VEL）和可评分的结构。
-   **可量化分数**：每个 Prompt 自带 `StructureScore` 和 `Q1-Q8` 多维评分，清楚知道哪里好、哪里该改。
-   **双门控判断**：AI 会直接告诉你这个 Prompt 是 `NoPost`（不可发布）还是 `PostReady`（已就绪）。
-   **一整套交付包**（使用 Seedance 时）：包括正片 Prompt、视觉增强编译版、安全降级版本等完整资产。

## 🤖 给 AI 的指令文件里到底有什么？

为了让使用者心里有数，这里简要拆解两份核心文档的模块：

### HappyHorse CreativeOS v2.1-MVP（通用技能包）
| 编号 | 模块 | 用途 |
|------|------|------|
| K1-K5 | 四元公式、模式路由、铁律 | 规范 Prompt 的基础要素生成 |
| K7 | VEL 视觉增强层 | 植入物理真实、角色生命、材质光影、大气粒子 |
| K8-K10 | Q1-Q8 评分、StructureScore | 给 Prompt 量化打分 |
| K18 | NoPost 可行性 | 判断能否交付 |
| 第六篇 | 14步审计法 | 质量校验 |
| 第八篇 | B5 校准矩阵 | 根据反馈迭代 |

### Seedance（专用编译器）
| 编号 | 模块 | 用途 |
|------|------|------|
| V5 | 6 步增强植入流程 | 从 RAW 到 Seedance 完整加工 |
| 第四层 | VEL 四层架构 | 精细化的视觉增强编译 |
| 降级总表 | 安全降级逻辑 | 保证输出不会崩坏 |

## ⚠️ 重要提醒

-   **所有评分均为专家先验，不是实际投放效果预测。** 只有接入真实生成数据并完成校准后，才可以讨论预测关系。
-   为了让 AI 完全遵循规则，**请务必将整个文档内容原样粘贴**，不要删减或总结。这些文档本身就是精心设计的 prompt 工程产物，任何修改都可能破坏其内部自检逻辑。
-   如果 AI 生成结果出现偏离，可以在对话中补充：“请重新按规范生成，并给出 StructureScore 评分”。
