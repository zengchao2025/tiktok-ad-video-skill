# tiktok-ad-video-skill

**Seedance/ HappyHorse Prompt 生成系统** —— 面向 TikTok 投放场景，集成创意搜索、结构评分、NoPost 直发优化与双门控审查，自动生成视频 Prompt、标题、标签、评论和 Bio，并提供风险、置信度与终止证明，帮助团队高效产出低后期依赖的增长素材。

此仓库包含两套独立且完整的 TikTok 视频 Prompt 生成系统：**Seedance** 和 **HappyHorse**。它们分别针对不同的平台与生产管线进行了深度适配。

| **版本** | v3.5 (Minor Release) | v1.0 (Major Release) |

### ✨ 关键交付物

每个系统在一次完整的运行后，会生成一份结构化的投放素材包，包含但不限于：

1.  **核心 Prompt**：可直接复制至对应 AI 平台的视频生成提示词。
2.  **投放文案 A/B/C**：提供反差、POV、情绪等不同风格的标题文案。
3.  **社交元数据**：包含 12-16 个标签、1 条置顶评论、1 条 Bio CTA。
4.  **质量评估报告**：涵盖结构质量评分、情绪影响快照 (Q8)、情绪弧密度 (EAD)、渲染置信度 (PRC) 等。
5.  **风险与证明**：提供“双门控”审查结果、有限终止证明及 VEL 植入记录，确保输出内容的合规性与高质量。

### 🚀 核心概念

#### 视觉增强层 (Visual Enhancement Layer, VEL)
一个统一的增强词库，分为**物理真实、角色生命、材质光影、大气粒子**四大类。在生成 Prompt 时，系统会自动遵循分散、不堆叠的植入规则，以提升画面的视觉表现力。

#### 双门控审查机制 (NoPostGate + PostReady)
一个模拟生产管线的质量关卡：
1.  **NoPostGate**：检查内容是否满足“无需后期制作即可发布”的直发标准。
2.  **PostReady**：评估作品是否已达到可投放的专业质量。

#### 评分体系
所有权重均基于**专家先验**设定，用于估算创意与结构的质量。项目声明此系统并非投放效果预测模型，所有权重有待数据校准。

### 📖 使用方法

1.  **准备输入**：准备好视频的核心创意、产品/USP、TikTok 趋势参考等信息。
2.  **选择系统**：
    *   若面向 **Seedance 2.0** 平台，请使用 `Seedance.md` 中的系统提示词。
    *   若面向 **HappyHorse 1.0** 平台，请使用 `HappyHorse.md` 中的系统提示词。
3.  **执行生成**：将对应文件的全部内容作为 System Prompt 提供给大语言模型，随后输入你的需求，即可获得完整的投放素材包。

> 提示：你可以直接使用主分支下相应的 `.md` 文件作为独立 Skill 进行加载和调用。
